# Place

Проект по обмену картинками и фотографиями

# Функционал

- добавление картинок
- удаление картинок
- просмотр картинки во всплывающем окне
- лайк картинки
- редактирование данных о пользователе

# Используемые технологии

- Javascript
- HTML
- CSS
- Webpack
